# Advanced Smart Contract Project

Смарт контракт за управление на токени и гласуване

## Технологии
- Solidity
- Web3.js
- Ethereum

## Инсталация
```bash
git clone https://github.com/LiliaPetrova/advanced-contract.git
cd advanced-contract/frontend
npm install -g http-server
http-server
```
